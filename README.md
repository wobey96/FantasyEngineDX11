# FantasyEngineDX11 
Fantasy Engine, a rendering engine with C++20 and DX11. Focused around Graphics Basics

## How to Build?
### Visual Studio (Windows)
Clone the project.
```
   git clone https://github.com/wobey96/FantasyEngineDX11.git
```
### CMake (Windows)

Open Visual Studio.
In Visual Studio, File -> New -> Project From Existing Code ...
